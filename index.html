Bem-vindo à API

Veja abaixo as rotas disponíveis.

Postagens

GET /postagem - Listar todas as postagens.
GET /postagem/:post_name - Buscar uma postagem por nome.
POST /postagem - Criar uma nova postagem.
PUT /postagem/:post_name - Editar uma postagem existente.
DELETE /postagem/:post_name - Deletar uma postagem.

Personagens

GET /personagem - Listar todos os personagens.
GET /personagem/:personagem_tag - Buscar um personagem por tag.
POST /personagem - Criar um novo personagem.
PUT /personagem/:personagem_tag - Editar um personagem existente.
DELETE /personagem/:personagem_tag - Deletar um personagem.

Usuários

GET /usuario - Listar todos os usuários.
GET /usuario/:uuid - Buscar um usuário por UUID.
POST /usuario - Criar um novo usuário.
PUT /usuario/:uuid - Editar um usuário existente.
DELETE /usuario/:uuid - Deletar um usuário.
