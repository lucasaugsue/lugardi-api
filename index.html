Bem-vindo à API

Veja abaixo as rotas disponíveis.

Assinaturas

GET /api/1/payment - Listagem dos pagamentos das assinaturas do EAD.
GET /api/1/signature - Listagem de assinaturas do EAD.
PUT /api/1/signature/{id} - Atualizar uma assinatura do EAD.

Banco de Questões

GET /api/1/questions - Listagem das questões cadastradas por curso.
POST /api/1/questions - Inserir uma questão para um curso.
PUT /api/1/questions/{pergunta_id} - Atualizar uma questão de um curso.
DELETE /api/1/questions/{id} - Excluir uma questão de um curso.
